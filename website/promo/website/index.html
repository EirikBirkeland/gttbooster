GTT Booster is a Chrome extension which greatly enhances Google Translator Toolkit. It is the result of years of working with GTT while refining and experimenting to come up with a toolkit that addresses its many lacks and shortcomings.

Features

  • Language checks
  • Improved glossaries
  • Spellchecking
  • Layout customization
  • Typing prediction
  • Keyword filtering
  • Tweaks & bugfixes
  • Embellishments
                                                                                             and more!

Save time and increase enjoyment

We've added a number of state-of-the-art tools to GTT: built-in language checks and the ability to add your own checks, spellchecking, typing autocompletion, and much more.

Language checks

You can even make your own tailored language checks* and share with your team. Templates and quick start guides are included.

Built-in checks include:
  • String-length validation
  • Leading and trailing spaces
  • Segment-level consistency check
  • Source equals target
  • and much more!
Core checks can be turned on/off and customized where applicable directly from the settings screen.

* Checks can be made using a simple, powerful syntax, and while it's completely optional and not essential, you can also leverage Regular Expressions, including the XRegExp library.

Options screen

Typing prediction

  • Words from the glossary are suggested automatically while typing.
  • Optionally, the TM can be leveraged as well.

Glossary-related

  • Glossary duplicates are now collapsed into single entries.
  • Entries are alphabetized.
  • Entries for the current product are highlighted.
  • New feature: You can also activate a separate glossary window which functions as a central server for any GTT tabs you are using. This "server" is always loaded with the latest glossaries.
    You may, for example, stick this window on a separate monitor, or just keep it as a semi-permanent fixture on the side.

Keyword filtering & segment type filtering

  • Type in a word or phrase, and display only the matching segments. Great for ensuring consistency!
  • You can hide any segment type you wish – for example ICE segments when you're looking over your own translation prior to delivery.

Other

  • Change text font
  • Hide segment descriptions
  • Full customization of all features through a comprehensive options screen.

A sneak peek at the toolbar itself

  • You get both always-on features and features that can be toggled. Toggleable features are manipulated through this toolbar. Full usage documentation is included!
